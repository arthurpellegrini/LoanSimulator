README

Présentation

Bienvenue dans le README de notre projet php. Ce projet répond au sujet suivant et consiste au développement d'un site de simulation de prêt bancaire. Ce site est "responsive design", c'est-à-dire que l'on peut réduire la fenêtre sans risque de manquement d'informations. Bien que notre site soit plus optimisé sur Firefox, nous avons essayé de faire en sorte que toutes fonctionnalités de la feuille de style puissent-être visibles sur tout type de navigateur.

Structure du site

Le schéma ci-dessous décrit le fonctionnement du site à haut niveau d'abstraction.

Usage

Dans cette partie, vous trouverez comment utiliser l'application que vous soyez un simple utilisateur ou l'administrateur du site.

Utilisateur

Pour effectuer une simulation, il suffit de vous rendre dans la page d'accueil, puis de vous rendre dans la section simulation. Dans celle-ci, vous aurez la possibilité de remplir les champs "capital", "taux" et "nombre de mois" qui sont nécessaires à la simulation. Vous retrouverez, ensuite votre simulation dans un tableau contenant l'historique des dix dernières simulations. Enfin, si vous souhaitez refaire une des simulations qui est présente dans le tableau, vous pouvez cliquer à l'aide de votre souris sur la ligne du tableau, et les champs nécessaires à la simulation se rempliront des valeurs présentes sur la ligne cliquée.

Administrateur

En plus de toutes les fonctionnalités présentées précédemment, l'administrateur peut en plus se connecter pour accéder au backend du site. Une fois connecté l'administrateur n'est pas obligé de se déconnecter pour accéder au reste du site. Depuis cette partie du site caché aux utilisateurs, l'administrateur a la possibilité de consulter, télécharger (archiver) et supprimer les logs ainsi que la totalité de l'historique des simulations.

Fonctionnement des fichiers

Dans cette partie, vous trouverez comment fonctionnent l'ensemble des fichiers php.

Partie Modèle/Contrôleur

Ces fichiers permettent le fonctionnement des différentes fonctionnalités du site.

connexion.php

Permet la connexion de l'administrateur au backend du site. Pour cela, il fait appel au fichier login.txt qui contient le nom d'utilisateur et le mot de passe hashé à l'aide de sha256, ceci lui permet de comparer ces données avec ceux entrés et ainsi de donner l'accès ou non au backend.

deconnexion.php

Permet la déconnexion de l'administrateur, c'est-à-dire la suppression du cookie de session.

historique.php

Contient la déclaration de deux fonctions, la première permet de lire le fichier historique.csv en retournant un tableau et la seconde permet d'insérer une nouvelle ligne au fichier historique.csv.

logs.php

Contient la déclaration de trois fonctions, la première permet de créer une variable de type string contenant les logs, la seconde permet d'ajouter du contenu au fichier logs et la troisième permet de lire le fichier logs.txt dans le but de pouvoir afficher les logs dans le site.

simulation.php

Permet une simulation à l'aide de trois paramètres : le capital, le taux et le nombre de mois. Celle-ci renvoie une valeur correspondante au montant de la mensualité. Dès lors qu'une simulation est effectuée, les données de celle-ci sont envoyées dans les fichiers historique.csv et logs.txt.

suppression.php

Permet la suppression du contenu du fichier historique.csv ou logs.txt suivant le nom de fichier passé grâce à la méthode get.

telechargement.php

Permet le téléchargement sur la machine de l'administrateur, du fichier historique.csv ou logs.txt suivant le fichier passé grâce à la méthode get.

Partie Vue

Dans cette partie, vous trouverez l'ensemble des fichiers qui permettent un affichage au site.

admin.php

Correspond au backend du site. Ce fichier fait appel à logs.php et à historique.php pour afficher les logs ainsi que l'historique des simulations, mais aussi à téléchargement.php et suppression.php pour télécharger ou supprimer le contenu des logs ou de l'historique.

connexion_form.php

Correspond au formulaire de connexion au backend du site. Ce fichier fait appel à connexion.php pour permettre la connexion.

index.php

Correspond à l'accueil du site sur lequel il est possible d'effectuer des simulations ainsi que de consulter l'historique des dix dernières simulations.
